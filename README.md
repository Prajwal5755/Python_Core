# Python Core Projects and Discord Bot

This repository contains my hands-on work with core Python concepts along with practical implementations including a Discord Bot using asyncio and data handling projects using MySQL, CSV, and file systems.

---

## Projects Overview

### 1. Discord Bot (Asyncio Based)

* Built using `discord.py`
* Uses Python’s `asyncio` for asynchronous execution
* Handles commands and events efficiently

#### Key Concepts:

* Async functions (`async` / `await`)
* Event-driven programming
* Non-blocking execution

---

### 2. Data Handling Applications

This section focuses on real-world Python usage with data processing and storage.

#### MySQL with Python

* Database connectivity using Python
* CRUD operations (Create, Read, Update, Delete)
* Query execution and result handling

#### CSV Handling

* Reading and writing CSV files
* Data processing using Python
* Structured data storage

#### File Handling

* Reading and writing files
* Working with text files
* File operations such as open, read, write, and append

---

## Tech Stack

* Python 3.x
* Libraries:

  * `asyncio`
  * `discord.py`
  * `mysql-connector-python`
  * `csv` (built-in)

---

## Example: Asyncio Usage

```python
import asyncio

async def main():
    print("Running async task")

asyncio.run(main())
```

## Environment Variables

Create a `.env` file:

```
